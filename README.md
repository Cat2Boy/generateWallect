## 普通钱包生成 

利用python 生成普通钱包 

依赖：
- web3py
- pandas

安装命令:
- pip install web3py

## HD钱包生成

[HD钱包原理介绍](https://github.com/inoutcode/ethereum_book/blob/master/%E7%AC%AC%E5%85%AD%E7%AB%A0.asciidoc)

HD钱包共用一套助记词，但是相互独立，某一个私钥丢失，不影响其他钱包

利用javascript

依赖：
- node-xlsx
- ethersjs

安装命令
- npm install node-xlsx
- npm install ethersjs
